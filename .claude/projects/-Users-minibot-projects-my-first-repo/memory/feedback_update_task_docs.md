name: Update task docs as steps complete
description: When working through multi-step tasks with a task doc, update the doc with results/details after each step completes — don't wait to be asked
type: feedback
---

When working through a multi-step task that has a task document (e.g. `.doc/task-*.md`), update the task doc automatically as each step completes. Mark the step as DONE, update the status line, and write up what was built/changed/learned. Don't wait for the dev to ask "is the task doc updated?" — just do it as part of finishing the step.
